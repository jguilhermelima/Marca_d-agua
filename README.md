<h1 align="center"> Desktop app imagem marca d'agua </h1>

<h2>Descrição do Projeto</h2>
Este código tem como funcionalidade adionar uma marca d'agua em uma imagem.

Passo a passo:

* Digitar a marca d'agua que deseja.
  
* Selecionar imagem salva no computador.

<h2>🛠️ Construído com</h2>

* Python

* Tkinter

* Pillow
